# CHANGELOG

## Ver3.0.6（開発中）
- 内部品質改善（計算ロジックの整理とテスト追加）
- UI調整
  - 必要エネルギー入力欄と投与量入力欄の相互同期を廃止し、必要エネルギー → 投与量の一方向同期に変更
  - 必要エネルギ-欄と経腸栄養投与量欄の手動編集ピルを独立化
  - 必要エネルギー側の手動編集ピルに「×」を表示し、自動計算に戻る導線を追加
  - 投与量入力欄を手動編集した場合は、経腸栄養欄内の計算結果に反映する（必要エネルギー入力欄は上書きしない）
  - スマホでの経腸栄養欄の折りたたみを廃止し、PCと同様に経腸栄養計算を始めたら常時表示に変更
